#  Page d'inscription Facebook (exercice HTML & CSS)

    Ce projet reproduit la page de création de compte Facebook à des fins d'apprentissage.  
    Il a été réalisé avec **HTML**, **CSS** 

## Structure du projet


    facebook-clone/

   index.html # Structure principale de la page
   facebook.css # Style complet (mise en page + responsive)
   README.md # Explication du projet
 

##  Fonctionnalités

   Formulaire d’inscription avec :
   Prénom, nom, email, mot de passe
   Choix du genre
   Message de succès après validation du formulaire
   Design inspiré de **Facebook**
   Version responsive (mobile )

##  Objectif du projet

  Projet réalisé pour **apprendre les bases du HTML et du CSS** à travers un exercice pratique.
